# UNERD Official Shop Store

E-Commerce store generated by STM Converter. Full-featured online store with seller dashboard, buyer registration, shopping cart, and Midtrans payment integration.

## Quick Start (Replit)

1. Import this repo into Replit
2. Add a PostgreSQL database from the Tools menu
3. Add API keys in the Secrets tab (see Payment & Shipping sections below)
4. Click Run - the app will auto-setup everything

## Quick Start (Local)

1. Set up PostgreSQL and create a database
2. The `.env` file will be auto-generated on first run
3. Edit `.env` to set your DATABASE_URL
4. Run: `bash start.sh`
5. Open http://localhost:5000

## Architecture

- **Frontend**: Next.js 14 (port 5000)
- **Backend**: Python FastAPI (port 8000)
- **Database**: PostgreSQL

## Payment (Midtrans)

To enable online payment, add these to your Secrets/Environment:
- `MIDTRANS_SERVER_KEY` - Get from Midtrans Dashboard > Settings > Access Keys
- `MIDTRANS_CLIENT_KEY` - Get from Midtrans Dashboard > Settings > Access Keys

For SANDBOX (testing): https://dashboard.sandbox.midtrans.com
For PRODUCTION: https://dashboard.midtrans.com (set `MIDTRANS_IS_PRODUCTION=true`)

Sandbox test card: `4811 1111 1111 1114`

## Shipping (Biteship)

To enable shipping cost calculation and courier selection, add:
- `BITESHIP_API_KEY` - Get from Biteship Dashboard > Settings > API Keys

1. Register at https://biteship.com
2. Go to Settings > API Keys in the dashboard
3. Create a new API key (use test/sandbox key for development)
4. Add it as `BITESHIP_API_KEY` in your Secrets/Environment

Supported couriers: JNE, SiCepat, J&T, AnterAja, TIKI, Ninja, ID Express, POS Indonesia

Features: Area search, multi-courier rate calculation, shipment creation, tracking, shipping labels
